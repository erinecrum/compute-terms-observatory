# Compute Terms Observatory — runtime dependencies (kept deliberately minimal,
# matching the radar project's low-dependency posture).
#
# Versions are PINNED exactly, not floored with >=. CI installs these into a job
# that holds ANTHROPIC_API_KEY and DATA_REPO_TOKEN, so a compromised release of
# any package here would run arbitrary code with the API key and with write access
# to the private corpus. Pinning makes an upgrade a reviewed commit rather than
# something that happens silently on the next scheduled run.
#
# Dependabot is enabled and will open PRs for security updates: bump these
# deliberately rather than loosening the pins back to >=.
